-------------------------------------------Manual de usuario--------------------------------------------------
1. Descargar e instalar Racket con el IDE DrRacket.

2. Clonar o descargar el proyecto desde el repositorio de GitHub.

3. Abrir el documento CatarCEQualifiers.rkt desde el IDE.

4. En la barra de tareas en la sección Lenguaje, seleccionar la opción de Teaching Languajes (ctl-T).

5.Determinar llamando la función principal desde el código con las formaciones y el número de generaciones o mediante la consola realizar el llamado.

6.Un ejemplo del llamado es (CatarCEQualifiers '(4 4 2) '(4 3 3) 4).

Durante la ejecución se puede observar como las generaciones realizadas por el algoritmo genético va haciendo que los jugadores busquen la bola y una vez la tienen disparen en dirección al marco rival.
Disfrute el juego mientras terminan la cantidad de generaciones que selecciono.
